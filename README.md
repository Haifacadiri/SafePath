# SafePath

📌 Project Overview

This project focuses on predicting the severity of traffic accidents using Machine Learning techniques. With the increasing number of road accidents causing injuries, fatalities, and economic losses, this system aims to provide a data-driven solution to analyze and predict accident severity.

The model helps in identifying high-risk conditions and supports better decision-making for accident prevention and emergency response.

🎯 Objective

Analyze environmental and road-related factors affecting accident severity

Build a Machine Learning model to predict accident severity

Provide insights to improve road safety and reduce severe accidents

📊 Dataset Overview

Dataset: Traffic Accident Severity Dataset

Key Features:

Weather Conditions

Road Surface

Light Conditions

Traffic Density

Target Variable:

Accident Severity (Fatal, Serious, Moderate, Minor)

⚙️ Approach
🔍 Data Preprocessing

Handled missing and inconsistent data

Performed feature encoding and transformation

📈 Exploratory Data Analysis (EDA)

Identified patterns and trends in accident data

Analyzed impact of weather, lighting, and road conditions

🤖 Model Building

Implemented multiple Machine Learning algorithms

Compared model performance using evaluation metrics

🏆 Best Model & Results

Best Model: XGBoost Classifier

Accuracy: 98%
